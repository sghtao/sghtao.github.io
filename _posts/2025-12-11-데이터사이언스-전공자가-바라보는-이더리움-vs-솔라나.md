학습 방향(솔리디티 vs 러스트)을 결정하려면, “블록체인 = 데이터베이스 + 실행환경(가상머신)” 프레임으로 EVM과 Solana(Sealevel/SVM)를 대비시키는 구성
DB 관점 → 아키텍처(EVM vs Sealevel) → 데이터 접근/분석 → 언어 선택

##블록체인이란?
#블록체인을 데이터베이스로 바라보기- 블록체인은 단순한 DB가 아니라, 합의(Consensus)가 병목인 분산 상태 머신(State Machine)이다
블록체인(blockchain)
분산 데이터베이스(Distributed Database)
분산 원장 기술(DLT, Distributed Ledger Technology)
state machine

비트코인 같은 블록체인 - distributed ledger
이더리움은 distributed state machine이다
Ethereum's state is a large data structure which holds not only all accounts and balances, but a machine state
